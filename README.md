# PRODIGY_ML_02
Implementation of K-Means clustering for retail customer segmentation using annual income and spending score features to identify distinct purchasing behavior groups.

## Dataset
Mall Customers Dataset (Kaggle)

## Features Used
- Annual Income (k$)
- Spending Score (1–100)

## Algorithm
- K-Means Clustering

## Output
A visual representation of customer groups based on purchasing behavior.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
